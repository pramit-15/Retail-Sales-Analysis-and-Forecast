🛒 Smart Retail Analytics & Demand Forecasting using Machine Learning
An End-to-End Retail Analytics Platform that performs sales analysis, customer insights, inventory intelligence, and demand forecasting using Machine Learning.
________________________________________
📌 Project Overview
Smart Retail Analytics & Demand Forecasting is a complete Machine Learning project developed to help retailers analyze historical sales data, identify business trends, understand customer purchasing behavior, and forecast future product demand.
The application provides an interactive web interface built with Flask where users can upload retail datasets, explore visual analytics, generate business insights, and predict future sales using trained machine learning models.
This project demonstrates the complete AI/ML lifecycle including:
•	Data Collection
•	Data Cleaning
•	Exploratory Data Analysis (EDA)
•	Feature Engineering
•	Machine Learning Model Training
•	Model Evaluation
•	Demand Forecasting
•	Data Visualization
•	Web Application Deployment
________________________________________

The primary objectives of this project are:
